TO INSTALL AUDITING TOOL

1. Download file from github and extract to folder.
2. Rename the file and remove the -master so its NETWORK-AUDITING-TOOL
3. Place folder into C: Directory on your computer.
4. Open it and click on Auditing_Tool.jar to run
(you can also copy this .jar onto your desktop to run there instead)

Report Scoring System
